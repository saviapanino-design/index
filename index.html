Sobre este ejemplo

Este es un archivo index.html básico que incluye las etiquetas fundamentales que toda página web debe tener:

  • DOCTYPE: Define la versión de HTML.
  • Head: Contiene metadatos y el título.
  • Body: Todo el contenido visible.

Puedes editar este archivo en cualquier editor de texto (como VS Code o Notepad++) y abrirlo en tu navegador favorito.
